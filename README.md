# Deliveroo app

My first React Native app, it was developed with react native as already said and styled with TailwindCss (NativeWind)

## `Libraries`

* react native
*@react-navigation
* react-redux
* @reduxjs/toolkit
* axios
* sanity
* tailwindcss
* nativewind
* react-native-animatable
* react-native-heroicons
* react-currency-formatter
* react-native-maps
